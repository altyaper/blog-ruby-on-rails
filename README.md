# News CMS

# Dependencies
Ruby 2.7.1
Rails 6.1.4.1
npm 6.14.6
node v12.18.3 (can be a lower version too)
PostgreSQL 13.3

# Install dependencies
```
bundle install
```
```
npm install
```

# Prepare database

```
rake db:setup
```

```
rake db:seed
```

# Run the app
```
rails server
```
